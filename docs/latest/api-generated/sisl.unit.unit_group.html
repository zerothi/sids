unit_group¶

sisl.unit.unit_group(unit, tbl=None)¶

The group of units that unit belong to

Parameters

  • unit (str) – unit, e.g. kg, Ang, eV etc. returns the type of unit it is.

  • tbl (dict, optional) – dictionary of units (default to the global table)

Examples

>>> unit_group('kg')
'mass'
>>> unit_group('eV')
'energy'
