Ellipsoid¶

class sisl.shape.Ellipsoid(v, center=None)[source]¶

3D Ellipsoid shape

Parameters:

v : float or (3,) or (3, 3)

radius/vectors defining the ellipsoid. For 3 values it corresponds to a Cartesian oriented ellipsoid.

center : (3,), optional

the center of the ellipsoid. Defaults to the origo.

Examples

>>> xyz = [0, 2, 0]
>>> shape = Ellipsoid([2, 2.2, 2])
>>> shape.within(xyz)
array([ True], dtype=bool)
