Module contents¶

Units from various programs

This conversion tool is inspired by the SIESTA fdf-parser in its group-construct.

sisl.units.unit_group(unit, tbl=None)[source]¶

The group of units that unit belong to

Parameters:

unit : str

unit, e.g. kg, Ang, eV etc. returns the type of unit it is.

tbl : dict, optional

dictionary of units (default to the global table)

Examples

>>> unit_group('kg')
'mass'
>>> unit_group('eV')
'energy'
sisl.units.unit_convert(fr, to, opts=None, tbl=None)[source]¶

Factor that takes ‘fr’ to the units of ‘to’.

Parameters:

fr : str

starting unit

to : str

ending unit

opts : dict, optional

controls whether the unit conversion is in powers or fractional units

tbl : dict, optional

dictionary of units (default to the global table)

Examples

>>> unit_convert('kg','g')
1000
>>> unit_convert('eV','J')
1.60219e-19
